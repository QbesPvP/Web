Normativas del Servidor

1. Normas Generales

1.1 Deathmatch DM

Esta prohibo disparar a alguien antes de darle un aviso de corte (Exceptuando cuando estamos dentro de una Zona Roja)

1.2 Corte s

Para poder comenzar a dispararle a alguien deberemos cortarlo y darle 3 segundos para que pueda reaccionar, esta totalmente prohibido cortar a alguien que se encuentre con las manos Levantadas.

1.3 Car Kill o VDM

Esta prohibido utilizar el auto para chocar o matar a cualquier persona.

2. Normas de Sistemas Y Animaciones

2.1 Animaciones

Esta Prohibido abusar de animaciones o darles un mal uso para Beneficio.

2.2 Bugs

El abuso de cualquier bug dentro del servidor conlleva a una sancion dentro del servidor.

2.3 Prendas

Esta prohibido el uso de cualquier prenda o pieza vestimenta que te haga tanquear (Cascos, gorras etc).

2.4 Dimension

Esta prohibido el uso del comando /dimension para abusar del mismo para un beneficio (Aparecer dentro de una zona etc).

2.5 Vestimenta

Esta prohibido utilizar la misma vestimenta con mas de 3 personas a la vez si no sos miembro de una mafia (Exceptuando eventos).

2.6 Tomar Altura

Esta completamente prohibido usar esta herramienta dentro del pvp para matar a otras personas o tomar altura para un beneficio.

2.7 No Props

Esta Prohibido utilizar NO PROPS para matar a traves de objetos/estructuras (En caso de usarlas para subir FPS esta bien, pero si se encuentra a alguien utilizandolas para matar a travez es BAN PERMANENTE).
